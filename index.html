Selected Publications

  • Exploring Quantum Machine Learning for PCOS Diagnosis: A Comparative Analysis, Springer, Apr 26, 2025.
    This study presents a comparative analysis of quantum-enhanced models (HQCNN, HQViT, VQAE, QSNN, and QGNN) for classifying PCOS medical images, with QSNN achieving the highest accuracy.
  • Comparative Analysis of Deep Learning Architectures for PCOS Detection: A Cross-Validation Approach, Springer, Feb 14, 2025.
    This research evaluates CNN, VGG16, EfficientNetV2B3, DenseNet121, ResNet50, and InceptionV3 for PCOS detection, addressing imbalance and overfitting via Stratified Cross-Validation.
  • Email Spam Detection using Amazon Web Services (AWS), Taylor & Francis, Jun 28, 2024.
    Proposes a cloud-based spam detection system using AWS SES, Lambda, and S3, achieving 97.22% accuracy with the BlazingText algorithm.
  • Dimensionality Reduction of KDD-99 using Self-Perpetuating Algorithm, Springer, May 29, 2021.
    Introduces a self-perpetuating algorithm to reduce KDD-99 dataset dimensions, improving accuracy and reducing features by 66.66% using J48.
